# Name Day API Documentation

Welcome to the Name Day API documentation. This API provides name day data for various months and countries. Name days are special days in some cultures where individuals celebrate their given names based on a calendar of saints or historical figures.

## Base URL
```
https://your-api-domain.com/
```

## Endpoints

### Get All Name Days
- **Endpoint**: `/`
- **Method**: `GET`
- **Description**: Retrieve name day data for all months and countries.
- **Example Request**:
  ```bash
  curl https://your-api-domain.com/
  ```
- **Example Response**:
  ```json
  {
    "january": {
      "1": {
        "hu": ["Fruzsina"],
        "pl": ["Mieczysław", "Mieczysława", "Mieszko"]
      },
      ...
      // More data for January
      ...
    },
    "february": {
      "1": {
        "hu": ["Ignác"],
        "pl": ["Dobrochna", "Iga", "Ignacy", "Paweł", "Siemirad", "Żegota"]
      },
      ...
      // More data for February
      ...
    },
    ...
    // More months
    ...
  }
  ```

### Get Name Days by Month
- **Endpoint**: `/:month`
- **Method**: `GET`
- **Description**: Retrieve name day data for a specific month.
- **Parameters**:
  - `month` (string): The name of the month in lowercase (e.g., "january").
- **Example Request**:
  ```bash
  curl https://your-api-domain.com/january
  ```
- **Example Response**:
  ```json
  {
    "1": {
      "hu": ["Fruzsina"],
      "pl": ["Mieczysław", "Mieczysława", "Mieszko"]
    },
    ...
    // More data for January
    ...
  }
  ```

### Get Name Days by Month and Date
- **Endpoint**: `/:month/:date`
- **Method**: `GET`
- **Description**: Retrieve name day data for a specific day within a month.
- **Parameters**:
  - `month` (string): The name of the month in lowercase (e.g., "january").
  - `date` (string): The day of the month as a number (e.g., "1" for the 1st day).
- **Example Request**:
  ```bash
  curl https://your-api-domain.com/january/1
  ```
- **Example Response**:
  ```json
  {
    "hu": ["Fruzsina"],
    "pl": ["Mieczysław", "Mieczysława", "Mieszko"]
  }
  ```

### Get Name Days by Month, Date, and Country
- **Endpoint**: `/:month/:date/:country`
- **Method**: `GET`
- **Description**: Retrieve name day data for a specific day within a month and a specific country.
- **Parameters**:
  - `month` (string): The name of the month in lowercase (e.g., "january").
  - `date` (string): The day of the month as a number (e.g., "1" for the 1st day).
  - `country` (string): The country code (e.g., "hu" for Hungary, "pl" for Poland).
- **Example Request**:
  ```bash
  curl https://your-api-domain.com/january/1/hu
  ```
- **Example Response**:
  ```json
  {
    "hu": ["Fruzsina"]
  }
  ```

## Response Format
- All responses are in JSON format.
- Successful responses return data corresponding to the requested parameters.
- If data is not found for the requested parameters, a `404` error response is returned with the message: `{ "error": "Data not found" }`.

## Error Handling
- The API will return appropriate HTTP status codes and error messages for invalid requests.
- Common error responses include:
  - `400 Bad Request`: Invalid request format or parameters.
  - `404 Not Found`: Data not found for the requested parameters.
  - `500 Internal Server Error`: Unexpected server errors.

## Rate Limiting
- This API may have rate limiting in place to prevent abuse. Refer to the API provider's documentation for rate limit details.
